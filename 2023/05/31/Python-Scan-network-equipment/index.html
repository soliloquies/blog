Python Scan network equipment

Python the paramiko module scans the in-network for the device.

Background

We are operating a big campus network, having many different vendor devices,
and building with different years.

Recently we found that many devices are out of management,
we need to rearrange all the devices in our network,
including management IP, for the manufacturer, model, device SN, and account password.

So, we need a program to scan the all networks.
Help us get it.

Needs list

  1. Try connecting the device for ssh or telnet.
  2. Can use different password groups to try it.
  3. Get the device manufacturer, model, system name, and device SN.
  4. Result from some information including running information, error, and basic file output.
  5. Give me an excel Database to me to manage and review it.

Code

Using the paramiko module, we program a class to connect the device for ssh.

It can connect devices, try multi-group accounts, and result in some information.
The re module, helps us filter just our needs information.

We create two lists, it save every thread result’s information.

final use the pandas module will all information save to a CSV file.
