Python for curl get pictures

Python analyer html code, and download pictures.

Design Logic

  1. Get HTML code with Chrome or Edge or another browser.
  2. Write the code into the file as save a html.txt.
  3. Python opens it and uses the open model to open it.
  4. Uses the re.findall method of the re module to find out what I need, such as title and pictures links.
  5. Create an execution function to download and save the file, and the parameter needs to be passed in as URL.
  6. End we can combine the various modules to use.
